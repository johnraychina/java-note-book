# JavaScript 介绍
核心语法（操作符、控制结构、语句）+ 标准库（Array，Date，Math等）
另外，还根据宿主环境会提供额外的API：浏览器、Node

对于浏览器：
- 浏览器控制类：操作浏览器
- DOM类：操作网页元素
- Web类：实现互联网的各种功能

对于Node：
- 文件操作API
- 网络操作API等
- USB访问、蓝牙等

Google Chrome App，Github Electron   
支持直接用JavaScript构建跨平台的桌面应用程序，不依赖于浏览器。

Google Chrome App甚至有一个叫做chrome dev editor开发编辑器.
